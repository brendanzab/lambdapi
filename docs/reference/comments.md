# Comments

## Line comments

Line comments are preceded by a double dash (`--`):

```pikelet
-- This is a comment!
```

> **Specification**:
>
> - [Surface language - Lexical syntax - Whitespace and comments][whitespace-and-comments]

## Doc comments

Documentation comments are preceded by three pipes (`|||`):

```pikelet
||| A doc comment!
```

> **Specification**:
>
> - [Surface language - Lexical syntax - Whitespace and comments][whitespace-and-comments]
